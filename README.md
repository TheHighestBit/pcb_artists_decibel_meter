# A NodeJS driver for PCB artists I2C Decibel Sound Level Meter Module

## Supported variants

| Variant | Board / SKU | Supported | Tested |
|---------|-------------|-------------:|------------------:|
| Regular | PCBA-DBM-r3 | :joy: | 1000 |
| Spectrum Analyzer    | PCBA-DBM-r3-SA | TBD | TBD |
| Pro     | PCBA-DBM-r3-PRO | TBD | TBD |

## Installation

```bash
npm install pcb-artists-decibel-meter
```

## Usage

```javascript
const { DecibelMeter } = require()
```
